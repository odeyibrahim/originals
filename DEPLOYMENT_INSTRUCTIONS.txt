========================================
V. GALLERY - DEPLOYMENT INSTRUCTIONS
========================================

Your Netlify-ready folder is at:
C:\Users\User\Desktop\vgallery-netlify-deploy

Or use the zip file at:
C:\Users\User\Desktop\vgallery-deploy.zip

TO DEPLOY TO NETLIFY:

Option 1 - Drag and Drop:
1. Go to app.netlify.com
2. Drag the FOLDER (C:\Users\User\Desktop\vgallery-netlify-deploy) directly onto Netlify
   (Not the zip file - drag the folder itself)

Option 2 - GitHub:
1. Push the contents of C:\Users\User\Desktop\vgallery-netlify-deploy to a GitHub repository
   (Make sure to push the FILES, not the parent folder)
2. Connect Netlify to your GitHub repo
3. Netlify will auto-detect the netlify.toml

Option 3 - Zip Upload:
1. Go to app.netlify.com
2. Drag the zip file (C:\Users\User\Desktop\vgallery-deploy.zip) onto Netlify
3. Netlify will extract and deploy

IMPORTANT: The root of your site must contain index.html
Current root files: .env.example, .gitignore, index.html, netlify.toml, package.json, README.md

ENVIRONMENT VARIABLES TO ADD IN NETLIFY:
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- PAYSTACK_SECRET_KEY
- ADMIN_PASSWORD (optional, default: admin123)

After deployment, your site will be live at:
https://your-site-name.netlify.app
